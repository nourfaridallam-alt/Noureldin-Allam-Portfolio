Project Overview

A virtual dice rolling simulator that generates random dice rolls with beautiful ASCII art visualization. The program simulates a standard six-sided die and displays the result using creative text-based graphics, making the digital dice rolling experience more engaging and visually appealing.

This project demonstrates fundamental programming concepts including random number generation, user input handling, control flow, and the creative use of data structures to store and display ASCII art patterns.

Challenges & Solutions

Challenge 1: ASCII Art Alignment

Problem: Creating visually consistent dice faces with proper spacing and alignment using only text characters.

Solution: Carefully designed each face as a list of strings with exact character counts, ensuring all faces maintain the same dimensions and borders align perfectly.

Challenge 2: Data Structure Organization

Problem: Efficiently storing and retrieving six different ASCII patterns.

Solution: Implemented a dictionary with integer keys (1-6) mapping to list values containing the ASCII art, enabling O(1) lookup time.

Challenge 3: User Input Handling

Problem: Managing various user inputs including valid choices, invalid entries, and case sensitivity.

Solution: Used the .lower() method for case-insensitive comparison and implemented comprehensive conditional logic with else clause for error handling.
