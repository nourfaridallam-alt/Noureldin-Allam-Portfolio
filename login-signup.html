Technical Implementation

File-Based Credential Storage

The application uses a text file (credentials.txt) to store user credentials in a pipe-separated format. On startup, the program loads all existing credentials into a dictionary for fast in-memory lookups during authentication.

base_dir = os.path.dirname(os.path.abspath(__file__))
file_path = os.path.join(base_dir, 'credentials.txt')

credentials = {}

with open(file_path, 'r') as f:
    for line in f:
        username, password = line.strip().split(" | ")
        credentials[username] = password

Login Authentication Logic

The login system validates user credentials by checking if the username exists in the credentials dictionary and if the password matches. Visual feedback is provided through color-coded labels.

def handle_login(event=None):
    username = username_entry.get()
    password = password_entry.get()

    if username in credentials and credentials[username] == password:
        message_label.config(text="Login successful!", fg="green")
    else:
        message_label.config(text="Invalid username or password", fg="red")

# Bind Enter key for quick submission
login_window.bind('', handle_login)

User Registration System

The signup functionality implements multiple validation checks: username uniqueness, password confirmation matching, and proper data formatting before writing to the credentials file.

def handle_signup(event=None):
    username = username_entry.get()
    password = password_entry.get()
    confirm = confirm_password.get()

    if username in credentials:
        message_label.config(text="Username already exists", fg="red")
    elif password != confirm:
        message_label.config(text="Passwords dont match, try again", fg="red")
    else:
        with open(file_path, 'a') as f:
            f.write(f"{username} | {password}\n")
        credentials[username] = password
        message_label.config(text="Sign-Up successful!!", fg="green")

Challenges & Solutions

Challenge 1: File Path Resolution

Problem: The credentials file needed to be accessible regardless of where the Python script was executed from.

Solution: Used os.path.dirname(__file__) to get the script's directory and os.path.join() to construct an absolute file path, ensuring the file is always found relative to the script location.

Challenge 2: Real-time Dictionary Updates

Problem: New registrations needed to be immediately available for login without restarting the application.

Solution: Maintained an in-memory dictionary that gets updated immediately after writing to the file, allowing instant authentication of newly registered users.

Challenge 3: Multiple Window Management

Problem: Creating separate windows for login and signup while maintaining the main window.

Solution: Used Tkinter's Toplevel() widget to create child windows that remain independent but associated with the root window, allowing multiple windows to coexist.

Challenge 4: Input Validation Feedback

Problem: Users needed clear visual feedback about validation errors and success states.

Solution: Implemented dynamic label updates with color coding (red for errors, green for success) using the config() method to modify label properties in real-time.

Complete Source Code

import os
import tkinter as tk

base_dir = os.path.dirname(os.path.abspath(__file__))
file_path = os.path.join(base_dir, 'credentials.txt')

credentials = {}

with open(file_path, 'r') as f:
    for line in f:
        username, password = line.strip().split(" | ")
        credentials[username] = password

root = tk.Tk()
root.title("Welcome user")
root.geometry("300x200")


def open_login():
    login_window = tk.Toplevel(root)
    login_window.title("Login")
    login_window.geometry("300x200")

    tk.Label(login_window, text="Username").pack()
    username_entry = tk.Entry(login_window)
    username_entry.pack()

    tk.Label(login_window, text="Password").pack()
    password_entry = tk.Entry(login_window, show="*")
    password_entry.pack()

    message_label = tk.Label(login_window, text="", fg="red")
    message_label.pack()

    def handle_login(event=None):
        username = username_entry.get()
        password = password_entry.get()

        if username in credentials and credentials[username] == password:
            message_label.config(text="Login successful!", fg="green")
        else:
            message_label.config(text="Invalid username or password", fg="red")

    login_window.bind('', handle_login)

    tk.Button(login_window, text="Login", width=15,
              command=handle_login).pack(pady=10)


def open_signup():
    signup_window = tk.Toplevel(root)
    signup_window.title("Sign Up")
    signup_window.geometry("300x250")

    tk.Label(signup_window, text="Username").pack()
    username_entry = tk.Entry(signup_window)
    username_entry.pack()

    tk.Label(signup_window, text="Password").pack()
    password_entry = tk.Entry(signup_window, show="*")
    password_entry.pack()

    tk.Label(signup_window, text="Confirm Password").pack()
    confirm_password = tk.Entry(signup_window, show="*")
    confirm_password.pack()

    message_label = tk.Label(signup_window, text="", fg="red")
    message_label.pack()

    def handle_signup(event=None):
        username = username_entry.get()
        password = password_entry.get()
        confirm = confirm_password.get()

        if username in credentials:
            message_label.config(text="Username already exists", fg="red")
        elif password != confirm:
            message_label.config(
                text="Passwords dont match, try again", fg="red")
        else:
            with open(file_path, 'a') as f:
                f.write(f"{username} | {password}\n")
            credentials[username] = password
            message_label.config(text="Sign-Up successful!!", fg="green")

    signup_window.bind('', handle_signup)

    tk.Button(signup_window, text="Sign Up", width=15,
              command=handle_signup).pack(pady=10)


tk.Button(root, text="Login", width=15, command=open_login).pack(pady=10)
tk.Button(root, text="Sign Up", width=15, command=open_signup).pack(pady=10)

root.mainloop()
