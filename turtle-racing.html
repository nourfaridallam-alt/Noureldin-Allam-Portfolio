Project Overview

An animated racing simulation game built with Python's Turtle graphics library. Players select the number of racers (2-10), and colorful turtle characters compete in a vertical race to the finish line. Each turtle moves forward random distances per iteration, creating an unpredictable and exciting race experience.

This project demonstrates mastery of graphics programming, modular function design, list operations, random number generation, and coordinate geometry.

Challenges & Solutions

Challenge 1: Equal Spacing Distribution

Problem: Needed to position turtles evenly across the screen width, regardless of how many participate.

Solution: Implemented dynamic spacing calculation dividing by (racers + 1) to ensure equal margins on both sides.

Challenge 2: Vertical Movement

Problem: Turtles default to facing right, but the race needs vertical upward movement.

Solution: Used racer.left(90) to rotate each turtle 90 degrees, changing their heading to point north.

Challenge 3: Identifying the Winner

Problem: Needed to determine which turtle object won and return its color.

Solution: Used colors[turtles.index(racer)] to find the winning turtle's position and return the corresponding color.

import turtle
import time
import random

# THESE ARE CONSTANT VALUES
WIDTH, HEIGHT = 500, 600
COLORS = ['red', 'green', 'blue', 'purple', 'pink',
          'cyan', 'yellow', 'orange', 'brown', 'black']


def get_number_of_racers():
    racers = 0
    while True:
        racers = input("how many turtles would you like to race? (2 - 10): ")
        if racers.isdigit():
            racers = int(racers)
        else:
            print("Input is not a number, Please enter a number!: ")
            continue

        if 2 <= racers <= 10:
            return racers
        else:
            print("Number is not between 2 - 10. Try Again!!")


def race(colors):
    turtles = create_turtles(colors)

    while True:
        for racer in turtles:
            distance = random.randrange(1, 20)
            racer.forward(distance)

            x, y = racer.pos()
            if y >= HEIGHT // 2 - 10:
                return colors[turtles.index(racer)]


def create_turtles(colors):
    turtles = []
    spacingx = WIDTH // (len(colors) + 1)
    for i, color in enumerate(colors):
        racer = turtle.Turtle()
        racer.color(color)
        racer.shape('turtle')
        racer.left(90)
        racer.penup()
        racer.setpos(-WIDTH//2 + (i+1) * spacingx, -HEIGHT//2 + 20)
        racer.pendown()
        turtles.append(racer)

    return turtles


def init_turtle():
    screen = turtle.Screen()
    screen.setup(WIDTH, HEIGHT)
    screen.title('turtle Racing!!')


racers = get_number_of_racers()
init_turtle()

random.shuffle(COLORS)
colors = COLORS[:racers]

winner = race(colors)
print("---------------------------------------------------")
print("The winner is the turtle with color: ", winner)
print("---------------------------------------------------")
time.sleep(5)
