Breadth-First Search Algorithm

The core of this project is the BFS algorithm, which explores the maze level by level, ensuring the first path found is always the shortest. The algorithm uses a queue (FIFO) to manage which cells to explore next, making it perfect for finding optimal paths.

Neighbor Detection

The algorithm explores all four cardinal directions (up, down, left, right) from each position, checking boundaries to avoid index errors and skipping walls marked with '#'.

def find_neighbors(maze, row, col):
    neighbors = []
    
    if row > 0:  # Up
        neighbors.append((row - 1, col))
    if row + 1 < len(maze):  # Down
        neighbors.append((row + 1, col))
    if col > 0:  # Left
        neighbors.append((row, col - 1))
    if col + 1 < len(maze[0]):  # Right
        neighbors.append((row, col + 1))
    
    return neighbors

How BFS Works

Breadth-First Search (BFS) is a graph traversal algorithm that explores nodes level by level. Here's how it solves the maze:

  1. Start at the initial position (marked with 'X')
  2. Add all reachable neighbors to a queue
  3. Mark current position as visited to avoid cycles
  4. Move to the next position in the queue
  5. Repeat steps 2-4 until the goal is found
  6. Reconstruct the path by backtracking through stored positions

Why BFS? Because it explores all positions at distance 'k' before moving to distance 'k+1', BFS guarantees finding the shortest path in an unweighted maze.

Challenges & Solutions

Challenge 1: Terminal Flickering

Problem: Initial attempts at animation caused screen flickering and poor visual experience.

Solution: Implemented the curses library's proper screen clearing and refresh cycle. Using stdscr.clear() followed by stdscr.refresh() ensures smooth, flicker-free updates.

Challenge 2: Path Reconstruction

Problem: BFS naturally finds if a path exists, but reconstructing the actual path required additional logic.

Solution: Stored the entire path history with each queued position. When a neighbor is added to the queue, the new path is created by appending the neighbor to the current path: new_path = path + [neighbor].

Challenge 3: Preventing Infinite Loops

Problem: Without tracking visited cells, the algorithm could revisit positions infinitely.

Solution: Implemented a visited set that tracks all explored positions. Before adding a neighbor to the queue, check if it's already been visited using if neighbor not in visited.

Challenge 4: Color Display in Terminal

Problem: Differentiating between the maze structure, current path, and walls visually.

Solution: Used curses color pairs to render the maze in blue and the active search path in red, making the algorithm's progress immediately visible.

def print_maze(maze, stdscr, path=[]):
    BLUE = curses.color_pair(1)
    RED = curses.color_pair(2)

    for i, row in enumerate(maze):
        for j, value in enumerate(row):
            if (i, j) in path:
                stdscr.addstr(i, j*2, "X", RED)
            else:
                stdscr.addstr(i, j*2, value, BLUE)

def find_start(maze, start):
    for i, row in enumerate(maze):
        for j, value in enumerate(row):
            if value == start:
                return i, j
    return None

def find_path(maze, stdscr):
    start = "X"
    end = "O"
    start_pos = find_start(maze, start)

    q = queue.Queue()
    q.put((start_pos, [start_pos]))
    visited = set()

    while not q.empty():
        current_pos, path = q.get()
        row, col = current_pos

        stdscr.clear()
        print_maze(maze, stdscr, path)
        time.sleep(0.2)
        stdscr.refresh()

        if maze[row][col] == end:
            return path

        neighbors = find_neighbors(maze, row, col)
        for neighbor in neighbors:
            if neighbor in visited:
                continue

            r, c = neighbor
            if maze[r][c] == "#":
                continue

            new_path = path + [neighbor]
            q.put((neighbor, new_path))
            visited.add(neighbor)

def find_neighbors(maze, row, col):
    neighbors = []

    if row > 0:  # up
        neighbors.append((row - 1, col))
    if row + 1 < len(maze):  # down
        neighbors.append((row + 1, col))
    if col > 0:  # left
        neighbors.append((row, col - 1))
    if col + 1 < len(maze[0]):  # right
        neighbors.append((row, col + 1))

    return neighbors

def main(stdscr):
    curses.init_pair(1, curses.COLOR_BLUE, curses.COLOR_BLACK)
    curses.init_pair(2, curses.COLOR_RED, curses.COLOR_BLACK)

    find_path(maze, stdscr)
    stdscr.getch()

wrapper(main)
