Curses Library for Terminal Graphics

The application uses Python's curses library for terminal manipulation, providing color support, positioning control, and keyboard input handling. The wrapper function ensures proper initialization and cleanup.

def main(stdscr):
    curses.init_pair(1, curses.COLOR_GREEN, curses.COLOR_BLACK)
    curses.init_pair(2, curses.COLOR_WHITE, curses.COLOR_BLACK)
    curses.init_pair(3, curses.COLOR_RED, curses.COLOR_BLACK)
    
    start_screen(stdscr)
    
    while True:
        wpm_test(stdscr)
        stdscr.addstr(2, 0, "You completed the text! Press any key to continue...")
        key = stdscr.getkey()
        if ord(key) == 27:  # ESC key
            break

wrapper(main)

The application calculates words-per-minute using the standard formula: (characters typed / 5) / (time in minutes). The calculation updates continuously as the user types, providing instant performance feedback.

Color-Coded Character Comparison

Each typed character is compared against the target text in real-time. Correct characters display in green using color pair 1, while incorrect characters show in red using color pair 3, providing immediate visual feedback.

def display_text(stdscr, target, current, wpm=0):
    stdscr.addstr(target)
    stdscr.addstr(1, 0, f"WPM: {wpm}")
    
    for i, char in enumerate(current):
        correct_char = target[i]
        color = curses.color_pair(1)  # Green
        if char != correct_char:
            color = curses.color_pair(3)  # Red
        
        stdscr.addstr(0, i, char, color)

Challenges & Solutions

Challenge 1: Screen Flickering

Problem: Continuous screen updates caused flickering and poor visual experience.

Solution: Used stdscr.clear() and stdscr.refresh() in proper sequence, along with the curses library's double-buffering capabilities to ensure smooth, flicker-free rendering.

Challenge 2: Backspace Key Handling

Problem: Different terminals represent backspace differently (KEY_BACKSPACE, '\b', "\x7f").

Solution: Checked for all three possible backspace representations to ensure cross-platform compatibility and consistent behavior across different terminal emulators.

Challenge 3: WPM Calculation Accuracy

Problem: Division by zero error when calculating WPM immediately after starting.

Solution: Used max(time.time() - start_time, 1) to ensure the elapsed time is never less than 1 second, preventing division by zero while maintaining calculation accuracy.

Challenge 4: Real-Time Updates Without Blocking

Problem: getkey() blocks execution, preventing continuous WPM updates.

Solution: Enabled non-blocking mode with nodelay(True) and wrapped getkey() in a try-except block to continue execution even when no key is pressed.

import curses
from curses import wrapper
import time
import random

def start_screen(stdscr):
    stdscr.clear()
    stdscr.addstr("Welcome to this typing test! :)")
    stdscr.addstr("\n-----------------------------------------------------------")
    stdscr.addstr("\nWould you like to play, Press a key to begin! :)")
    stdscr.refresh()
    stdscr.getkey()

def display_text(stdscr, target, current, wpm=0):
    stdscr.addstr(target)
    stdscr.addstr(1, 0, f"WPM: {wpm}")
    
    for i, char in enumerate(current):
        correct_char = target[i]
        color = curses.color_pair(1)
        if char != correct_char:
            color = curses.color_pair(3)
        
        stdscr.addstr(0, i, char, color)

def load_text():
    with open("text.txt", 'r') as f:
        lines = f.readlines()
        return random.choice(lines).strip()

def wpm_test(stdscr):
    target_text = load_text()
    current_text = []
    wpm = 0
    start_time = time.time()
    stdscr.nodelay(True)
    
    while True:
        time_elapsed = max(time.time() - start_time, 1)
        wpm = round((len(current_text) / (time_elapsed / 60)) / 5)
        
        stdscr.clear()
        display_text(stdscr, target_text, current_text, wpm)
        stdscr.refresh()
        
        if "".join(current_text) == target_text:
            stdscr.nodelay(False)
            break
        
        try:
            key = stdscr.getkey()
        except:
            continue
        
        # Ordinal value of the key esc
        if ord(key) == 27:
            break
        
        if key in ("KEY_BACKSPACE", '\b', "\x7f"):
            if len(current_text) > 0:
                current_text.pop()
        elif len(current_text) < len(target_text):
            current_text.append(key)

def main(stdscr):
    curses.init_pair(1, curses.COLOR_GREEN, curses.COLOR_BLACK)
    curses.init_pair(2, curses.COLOR_WHITE, curses.COLOR_BLACK)
    curses.init_pair(3, curses.COLOR_RED, curses.COLOR_BLACK)
    
    start_screen(stdscr)
    
    while True:
        wpm_test(stdscr)
        stdscr.addstr(2, 0, "You completed the text! Press any key to continue...")
        key = stdscr.getkey()
        if ord(key) == 27:
            break

wrapper(main)
